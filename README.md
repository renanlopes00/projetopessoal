O CodeCursos é uma plataforma fictícia de cursos online desenvolvida para aplicar os conhecimentos de HTML5 e CSS3. O objetivo é oferecer uma interface simples, organizada e intuitiva para usuários que desejam aprender programação.

Objetivo

Criar um site de cursos com uma navegação simples, permitindo que o usuário conheça diferentes áreas da programação e acesse informações sobre cada curso.

Público-Alvo
Iniciantes na programação.
Estudantes.
Pessoas interessadas em tecnologia e desenvolvimento web.
Funcionalidades
Página inicial com apresentação da plataforma.
Seleção de categorias de cursos.
Cards com os principais cursos disponíveis.
Página individual para cada curso.
Página de login.
Página de contato.
Menu de navegação entre todas as páginas.
Rodapé com informações da plataforma.

Estrutura do Site

O projeto é composto por várias páginas interligadas:

Página Inicial: apresenta a plataforma e os cursos.
Página de Cursos: exibe os cursos disponíveis.
Página de HTML, CSS, JavaScript e Python: apresenta informações específicas sobre cada curso.
Página de Login: permite que o usuário acesse sua conta.
Página de Contato: oferece um formulário para envio de mensagens.
Recursos Utilizados
Menu de navegação.
Imagens representando cada curso.
Cards organizados com Grid.
Cabeçalho e rodapé utilizando Flexbox.
Formulários.
Botões interativos com efeito hover.
Organização Visual

O site utiliza uma identidade visual baseada em tons de azul e branco, transmitindo uma aparência moderna, organizada e agradável para o usuário.
